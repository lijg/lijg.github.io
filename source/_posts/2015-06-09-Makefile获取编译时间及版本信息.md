title: Makefile获取编译时间及版本信息

categories: Linux
---


在`Makefile`中获取编译时间及版本信息，

在`Makefile`中添加以下语句,

```
# 编译时间
COMPILE_TIME = $(shell date +"%Y-%M-%d %H:%M:%S")
CFLAGS += "\"$(COMPILE_TIME)\""

# SVN版本号
SVN_REVIVION = $(shell svnversion -n)
CFLAGS += "\"$(SVN_REVIVION)\""

# GIT版本号
GIT_REVISION = $(shell git show -s --pretty=format:%h)
CFLAGS += "\"$(GIT_REVISION)\""
```
