#0. 处理UTF-8
默认情况下python是不能处理中文，想要处理中文必须使用utf-8编码，如
```
#!/usr/bin/python
# -*- coding: utf-8 -*- 

print "中文打印"
```

#1. 注释

* python只有单行注释，使用#号，如
```
# 这里是注释
# 打印提示信息
print "Comment Example."
```

* 如果需要多行注释，可以使用定义多行文本的方法，如
```
'''
这里是注释
打印提示信息
'''
print "Comment Example."
```

* 添加注释的原则:

**Code tells you how, comments should tell why.**
